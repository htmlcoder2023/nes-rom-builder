# nes-rom-builder

This is a Python program that attempts to build matching game ROMs by brute-forcing every possible ROM combination for a certain ROM size.

# How to Run

* Windows
  
  1. Download the latest release (v1.3 at the time of this writing) and extract the folder nes-rom-builder-exe.
  2. Tap on the .exe file to run.

OR

  1. Install Python 3.12.4 or auto-py-to-exe.
  2. Run `cd Downloads`.
  3. Run `git clone https://github.com/htmlcoder2023/nes-rom-builder.git`.
  4. Run `cd nes-rom-builder/nes` or `cd nes-rom-builder/fds`.
  5. Run `[SCRIPT]`.

* Linux

  1. Run `pkg install python`. This should install the latest version of Python.
  2. Install git on Termux using this guide https://www.geeksforgeeks.org/how-to-install-git-on-termux/.
  3. Run `cd ~/storage/downloads`.
  4. Run `git clone https://github.com/htmlcoder2023/nes-rom-builder.git`.
  5. Run `cd nes-rom-builder/nes` or `cd nes-rom-builder/fds`.
  6. Run `python [SCRIPT]`.
